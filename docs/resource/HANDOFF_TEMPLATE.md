■ 사용법 (둘 중 하나)

     (스킬 위치: .claude/skills/handoff/SKILL.md)

■ 핵심 원칙 (이게 인계 문서의 가치를 만든다)
  - ✅ 는 "검증된 것만". 각 항목에 증거(`실행한 명령` · `파일:라인` · 테스트 결과)를 붙인다.
    증거가 없으면 ✅ 가 아니다 → ⏳ 로 내린다. "되겠지"로 완료 표시 금지.
  - ❌ 는 "시도했지만 안 된 접근 + 원인". 다음 세션이 같은 막다른 길을 반복하지 않게.
  - ⏳ 는 "바로 이어갈 다음 단계 + 시작점(파일/명령)". 미착수 항목도 빠짐없이.
  - 이어받는 사람 관점으로 쓴다: "내가 뭘 했나"가 아니라 "다음 사람이 뭘 알아야 이어가나".
  - 간결·스캔 가능하게. 각 항목 한 줄. 산문 단락 금지.
  - 기존 인계 파일이 있으면 덮어쓰지 말고 갱신: 끝난 항목은 ✅로, 유효한 미완료는 ⏳로 carry-forward.
